# 🍋 Lemonade App

A simple and interactive Android app built using **Kotlin** and **Jetpack Compose** that simulates the process of making lemonade — from picking a lemon to drinking it!

---

##  Features

* 🍋 Tap to pick a lemon from the tree
* ✋ Squeeze the lemon multiple times
* 🥤 Drink the lemonade
* 🔁 Restart the process

---

##  Built With

* **Kotlin**
* **Jetpack Compose**
* **Android Studio**

##  How It Works

The app follows a simple state-based flow:

1. Select a lemon from the tree
2. Squeeze it (random number of times)
3. Drink the lemonade
4. Restart

Each tap changes the UI dynamically using Compose state management.
